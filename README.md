# Customer-Behavior-Simulation-Dynamic-vs.-Aggressive-Marketing-Strategies
Экспериментальное исследование влияния различных стратегий коммуникации на долгосрочную прибыль. С помощью симуляции на синтетических данных сравниваются два подхода: агрессивный маркетинг и динамическое взаимодействие.

## Реализованный функционал
* Генерация синтетических данных для моделирования отклика потребителей
* Создание механизма «усталости клиента», который снижает вероятность покупки при избыточном давлении
* Сравнение агрессивной стратегии и динамического подхода
* Оценка влияния учета состояния клиента на итоговый финансовый результат

## Выводы исследования
Симуляция показала преимущество динамического взаимодействия. Учет фактора накопленной усталости позволил увеличить долгосрочную прибыль на пользователя примерно на один процент.

## Технологический стек
* Python для разработки логики симулятора
* NumPy и SciPy для математических вычислений и статистики
* Scikit-learn для базовой классификации состояний
* Pandas для формирования отчетов по результатам тестов
